Sync Dropbox photos to Flickr and G+ easily. You can sync manually whenever you want or use the settings to make it works automatically. We also provide a CLI program so you don't have to install the whole packages, open the browser and connect to the web page. I think this woule be very convinient to some programers who works with CLI environment or terminal.

PhoSync is an open source project. We make it because we don't like the design of other similar providers. You can download it at github and install it on your own servers or devices. If you don't like the current design, you can fork it and modify it to what you like by yourself.
